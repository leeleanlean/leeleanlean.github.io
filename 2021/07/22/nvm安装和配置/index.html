nvm安装和配置

1. 下载&安装

1. 下载地址
https://github.com/coreybutler/nvm-windows/releases

2. 下载文件
nvm-setup.zip

3. 解压、安装 - * 记好安装nvm的目录

2. 安装指定版本Node.js

由于nvm use之后，某些版本没有带npm，所以便捷安装方式如下：

2.1 下载指定版本的node.js包

https://npm.taobao.org/mirrors/node/

// 选择指定版本，如: v10.15.0

// 下载指定的包，如: node-v10.15.0-win-x64.zip

2.2 解压下载的包

2.3 修改包的名称

如 v10.15.0

2.4 拷贝到nvm安装目录下

3. 卸载指定版本node.js

进入nvm安装目录，删除指定版本的node.js安装目录。如: v10.15.0

4. 使用nvm

// 列出已安装的node.js版本
nvm list

// 使用指定版本的node.js
nvm use 10.15.0

// 查看正在使用的node.js版本
node -v

// 查看正在使用的npm版本
npm -v

5. Enjoy
